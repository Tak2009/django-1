🐱 🐱Cat Monitor: We used to have mice issues in our garage. Thus I built an auto motion detection night vision device with Raspberry Pi and Python to detect mice during night time. Now we have a cat and all the issues are gone!! Thus, I have decided to use the device not only to monitor the cat but also keep the photo records as her own "blog posts". She is a lovely cat but we do not know what she is doing during night time. Once a motion is detected by the device, the device send the pic (as a hard evidence!) to the web server via HTTP POST request and the Cat Monitor menu can show whatever she does, her secret night life!!
